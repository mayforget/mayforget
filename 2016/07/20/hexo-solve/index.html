Hexo搭建常见问题及解决方案总结

发表于 2016-07-20   |     |   阅读次数

上一篇文章介绍了Hexo+Coding搭建Hexo博客的主要步骤，搭建完成后出现了自己想看到的界面，但是官方的提供主题并不能满足我们各种各样的需求，这个给出一个主题归纳链接，里面总结了很多主题可供选择，具体使用可参考对应主题的官方文档。本篇文章主要介绍的是Hexo搭建过程中常见的问题及经过多次查询尝试后测试有效的解决方案。

1、输入sudo npm install -g hexo命令时出现以下警告信息

fsevents@1.0.12 install /usr/local/lib/node_modules/hexo/node_modules/chokidar/node_modules/fsevents
node-gyp install --fallback-to-buid

gyp WARN EACCES user "root" does not have permission to access the dev dir >"/Users/xiangwenwen/.node-gyp/6.2.2"
gyp WARN EACCES attempting to reinstall using temporary dev dir >"/usr/local/lib/node_modules/hexo/node_modules/chokidar/node_modules/fsevents/.node-gyp"

解决方案：

尝试sudo npm install -g -unsafe-perm hexo无效，然后尝试sudo chmod 777 /var/root依然无效，最后打开VPN再次输入sudo npm install -g hexo竟然解决了。

2、输入npm install hexo -g安装命令出现以下错误提示

{ [Error: Cannot find module './build/Release/DTraceProviderBindings'] code: 'MODULE_NOT_FOUND' }  
{ [Error: Cannot find module './build/default/DTraceProviderBindings'] code: 'MODULE_NOT_FOUND' }  
{ [Error: Cannot find module './build/Debug/DTraceProviderBindings'] code: 'MODULE_NOT_FOUND' }

解决方案：

npm install hexo --no-optional

3、hexo d上传出现spawn git ENOENT错误提示，详情如下

Error: spawn git ENOENT
   at exports._errnoException (util.js:746:11)
   at Process.ChildProcess._handle.onexi
   (child_process.js:1053:32)
   at child_process.js:1144:20)
   at process._tickCallback (node.js:355:11)

解决方案：

\Git\bin;\Git\libexec\git-core添加Git环境变量即可

4、MAC 下找不到id_rsa.pub和id_rsa文件

解决方案：

Finder里按下cmd+shift+G后输入~/.ssh查找即可看到

5、想重新安装Hexo或者安装旧版本，怎么卸载Hexo

解决方案：

npm uninstall hexo-cli -g
